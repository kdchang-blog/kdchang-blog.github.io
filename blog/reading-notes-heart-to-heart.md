# 本書金句

> 「在複雜又紛亂的世界裡，再微小的良善，都是劃破黑夜的一束光。真正的平靜不是世界安靜，而是心能安住於世界的喧囂中。」

# 前言

在當前這個複雜且充滿變動的世界中，我們需要一股溫暖而堅定的力量來指引方向。《從心開始》這本圖文書，匯集了全球心靈導師達賴喇嘛的智慧與暢銷繪本作家麥當諾的溫暖畫筆，共同為讀者獻上一趟發人深省的旅程。這不僅是一本探討環境保護的書籍，更是一次對內在的深刻檢視。透過大貓熊與達賴喇嘛的對話，它提醒我們，真正的平靜與喜悅，來自於擺脫過度的自我關注與負面思想，並強調「心」的轉變是推動世界朝向正確方向前進的根本力量。這本書以可愛的圖文，傳達了跨越年齡的普世價值：再微小的良善，都能成為劃破黑夜的光束，讓我們從心開始，擁抱彼此，共創一個更美好的大家庭。

![](/img/posts/reading-notes-heart-to-heart-2.jpg)

# 重點摘要

1.  良善是劃破黑夜的光：強調在複雜紛亂的世界中，微小的良善也能帶來巨大的正向影響。
2.  快樂的真諦：世間所有快樂源於希望他人快樂；所有苦惱源於追求自我快樂。
3.  活在當下：告誡人們只有「今天」可以行動，不被昨天與明天所束縛。
4.  信仰的核心：真正的信仰不需要複雜的哲理或寺廟，而是「盡可能善待他人」。
5.  內在的檢視與改變：鼓勵讀者擺脫習慣性的過度自我關注及負面思想，以找到真正的平靜與喜悅。
6.  普世責任：達賴喇嘛主張每個人對環境保護負有責任，需為未來世代留下一個安全的世界。

# 應用行動方案

1.  每日實踐一個「盡可能善待他人」的具體行動，無論是對家人、同事或陌生人。
2.  每天撥出固定時間進行內在檢視，覺察並調整過度的自我關注或負面思想。
3.  將注意力從「我想要」轉向「我能為他人做什麼」，從而培養希望他人快樂的心態。

# 問題討論

1.  如何平衡對自己快樂的追求與希望他人快樂的願望？
2.  在日常生活中的挑戰和壓力下，如何有效擺脫習慣性的負面思想？
3.  對於「盡可能善待他人」這一理念，在現代社會中可能遭遇哪些實踐上的困難？

# 總結

《從心開始》是一本結合智慧與溫暖的啟發之作。它以簡單卻深刻的圖文，清晰地傳達了達賴喇嘛關於愛、同情和普世責任的訊息。本書的核心理念是，唯有從我們的心靈深處做出改變，才能真正影響周遭的世界，進而讓人類這個大家庭邁向更光明的未來。它肯定了每個人的個人力量，鼓勵我們以良善和同理心作為生活的指引。這本書是給予所有年齡層的一份肯定生命的禮物，指引我們如何透過「心」的轉變，成為地球未來的守護者。
